# TicTacToe
  
# Installation  
$ git clone https://github.com/mjankovskij/TicTacToe.git  
$ cd TicTacToe  
$ composer require symfony/webpack-encore-bundle  
  
# Run Using Symfony Server  
$ symfony server:start
And just visit the url: http://localhost:8000  
  
# Test  
$ bin/phpunit
